# AniWorld Android Client 🚀

An unofficial, work-in-progress Android client for AniWorld. This project aims to provide a native viewing experience by integrating a custom video player directly into the application.

## ⚠️ Status: Beta (Under Development)
**Warning:** This project is in an early stage of development. It is currently **not intended for daily use** and may contain bugs, instabilities, or incomplete features.

## Features
*   **Integrated Browsing:** Access AniWorld content directly within the app.
*   **Custom Video Player:** A dedicated, native player designed for an optimized viewing experience.
*   **In Development:** Additional features like watch history and download managers are planned for future updates.

## Installation
You don't need to compile the source code yourself. 

1. Go to the [Releases page](https://github.com/hakuxyz/aniworld-viewer/releases).
2. Download the latest `.apk` file.
3. Install the APK on your Android device (make sure to allow "Install from unknown sources" in your settings).
